<img src="https://img.shields.io/badge/Status-Production_Ready-34D399?style=flat-square"/> <img src="https://img.shields.io/badge/Type-Backend_System-38BDF8?style=flat-square"/> <img src="https://img.shields.io/badge/Focus-Security_%26_Scalability-818CF8?style=flat-square"/>

A fully production-ready authentication and authorization backend — designed from the ground up with security, modularity, and real-world usage in mind.

### 🤖 IntelliBot — `Hybrid AI Orchestration`

<img src="https://img.shields.io/badge/Status-Active-34D399?style=flat-square"/> <img src="https://img.shields.io/badge/Type-AI_%2B_Backend-818CF8?style=flat-square"/> <img src="https://img.shields.io/badge/Pattern-Offline_%2B_Online_AI-F59E0B?style=flat-square"/>

> **Not just a chatbot — a backend orchestration pattern for AI systems.**

A hybrid AI system that intelligently routes between offline ML inference and online LLM APIs, balancing cost, latency, and accuracy.

**Architecture Pattern:**

```
User Query
    │
    ▼
Intent Classifier (offline TF-IDF + ML)
    │
    ├── High confidence → Local response (fast, free)
    │
    └── Low confidence → LLM API (accurate, online)
```

**Stack:** `Python` `NLP` `TF-IDF` `LLM APIs` `Backend Orchestration`
